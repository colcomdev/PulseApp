To start a Django project, you need to set up your environment by creating and activating a virtual environment, installing Django using pip, creating the project using django-admin startproject, and then running the development server. 
Here are the detailed steps to get your Django project running:
Prerequisites
Python 3: Ensure you have Python installed. You can download the latest version from Python.org.
Command Line Interface: Familiarity with your system's command line (Terminal, Command Prompt, or PowerShell) is needed. 
Step-by-Step Guide
Create a Project Directory
Open your terminal and create a new folder for your project, then navigate into it:
bash
mkdir my_django_project
cd my_django_project
Set Up and Activate a Virtual Environment
Using a virtual environment is recommended as it isolates project-specific packages from your system-wide Python installation.
Create the environment:
bash
python -m venv .venv
Activate the environment:
macOS/Linux:
bash
source .venv/bin/activate
Windows (Command Prompt):
bash
.venv\Scripts\activate.bat
Windows (PowerShell):
powershell
.venv\Scripts\Activate.ps1
Once activated, you should see the name of your environment (e.g., (.venv)) at the beginning of your terminal prompt.
Install Django
With your virtual environment active, install Django using the Python package manager pip:
bash
pip install django
You can verify the installation by checking the version:
bash
python -m django --version
(Optional) Pin your dependencies to a requirements.txt file for easy recreation later:
bash
pip freeze > requirements.txt
Create the Django Project
Run the django-admin command to bootstrap the project structure. The dot (.) at the end of the command places the project files in your current directory, avoiding an extra nested folder.
bash
django-admin startproject mysite .
Run the Development Server
Navigate to your project directory (if you used the . in the previous step, you are already in the correct spot) and start the local development server:
bash
python manage.py runserver
Verify the Installation
Open your web browser and navigate to the address shown in the terminal output (usually http://127.0.0.1:8000/ or http://localhost:8000/). You should see a default Django welcome page, indicating your project is successfully set up. 
Next Steps: Create a Django App 
A Django project is a collection of settings, while an "app" is a web application that does something specific (e.g., a blog, a poll app). To start building functionality, you create an app: 
bash
python manage.py startapp myapp
This command creates a myapp folder, which is where you will define models, views, and other application logic.
